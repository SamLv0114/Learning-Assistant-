# ML Learning Assistant

A two-mode AI-powered learning assistant that provides personalized ML research recommendations and answers questions from your knowledge base.

## Features

### Mode 1: Daily Feed (Proactive)
- **Automated Daily Pipeline**: Collects new papers and articles every morning
- **Classical ML Ranking**: Uses Gradient Boosting to rank content by relevance
- **Personalized Summaries**: LLM-generated summaries tailored to your interests
- **Smart Filtering**: Only shows high-quality, relevant content

### Mode 2: Q&A Assistant (Reactive)
- **Vector Search**: Semantic search across your knowledge base
- **RAG-based Answers**: Generates answers with source citations
- **Context-Aware**: Understands your questions and provides detailed explanations
